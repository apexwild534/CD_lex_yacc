# LEX and YACC codes

## Installation

To set up Lex and Yacc (Flex and Bison), follow the instructions below:

   ```bash
   sudo apt-get update
   sudo apt-get upgrade
   sudo apt install flex
   sudo apt install bison
   ```
## To the Output
Press ctrl+d
